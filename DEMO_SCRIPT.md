# YieldNet Demo Video Script (~2 min)

Screen recording + English voiceover. No need to show face.

---

## 1. Opening (10s)

> "Hi, I'm [your name]. This is YieldNet — a cross-chain yield net value calculator built on LI.FI APIs."

- Show: homepage hero section

## 2. Problem (15s)

> "DeFi yields look great on paper, but cross-chain costs like gas, bridge fees, and slippage eat into your returns. There's no tool that shows you the real net profit after these hidden costs."

- Show: scroll to Problem vs Solution cards on homepage

## 3. Explore (20s)

> "YieldNet pulls real vault data from the LI.FI Earn API. You can filter by chain, protocol, or yield type — lending, staking, farming, and more."

- Show: open Explore page
- Show: click yield type filter, switch between types
- Show: click a high-APY vault

## 4. Calculator — Live Quotes (30s)

> "The calculator fetches real-time cross-chain cost quotes using the LI.FI Composer API — not hardcoded estimates."

- Show: select source chain (Ethereum) and destination chain (Arbitrum)
- Show: green "Live Quote" badge appears
- Show: gas cost, bridge fee, slippage breakdown

> "Change the amount, holding period, or chains — the net yield updates instantly."

- Show: adjust amount slider and holding days
- Show: net yield result (green = profitable / red = unprofitable)
- Click "View Dashboard"

## 5. Dashboard (20s)

> "The dashboard visualizes your yield over time, break-even crossover, and how deposit size affects your net APY."

- Show: scroll through 4 charts
- Show: point out the break-even reference line

> "Smaller deposits take longer to break even because fixed bridge costs take a bigger percentage."

## 6. API Integration (15s)

> "Under the hood, YieldNet integrates two LI.FI APIs. The Earn API for vault discovery across multiple chains, and the Composer API for real-time bridge and gas cost quotes. The quote proxy runs server-side to keep the API key secure."

- Show: briefly show the architecture (optional: quick flash of code)

## 7. Closing (10s)

> "YieldNet helps DeFi users make informed decisions by showing the true net value of cross-chain yields. Try it at yieldnet.vercel.app. Built for the DeFi Mullet Hackathon. Thank you."

- Show: homepage with URL visible

---

## Tips

- Keep it under 2 minutes
- Use a clean browser (no extra tabs/bookmarks)
- Switch to dark mode for better visual impact
- Record at 1080p or higher
- Speak slowly and clearly
- Tools: OBS Studio (free), QuickTime (Mac built-in), or Loom
